# Quick Journal

用自然语言快速记录你的日常，AI 帮你整理成结构化日记。

## 使用方式

直接说出你想记录的内容即可：

> "今天开了好多会，人快麻了，晚上去跑了 5 公里，好多了"

> "@quick-journal 记一下：买了一本《原则》，准备这周开始读"

## 支持的命令

| 说法 | 功能 |
|------|------|
| "记录一下..." | 生成当日条目 |
| "生成本周总结" | 汇总本周日记 |
| "帮我改一下语气" | 调整条目风格 |
| "加个标签：..." | 追加自定义标签 |

## 数据格式
所有日记条目为标准 Markdown，存储在你本地，AI 不上传任何内容。

## 自定义
在 `resources/mood-tags.json` 中可以添加你自己的情绪标签和别名。
