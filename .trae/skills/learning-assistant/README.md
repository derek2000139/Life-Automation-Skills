# Learning Assistant

把知识变成真正属于你的东西。用闪卡、费曼学习法和科学复习计划，帮你理解和记住。

## 使用方式

| 需求 | 说法 |
|------|------|
| 做闪卡 | "帮我学 JS 闭包，做 5 张闪卡" |
| 费曼解释 | "用费曼法解释什么是闭包" |
| 复习计划 | "帮我生成艾宾浩斯复习计划" |
| 综合学习 | "给我一份关于 Python 列表的学习材料" |

## 学习工具

- 闪卡：一卡一问，难度分级，考察真正理解
- 费曼学习单：用大白话和类比解释复杂概念
- 复习计划：基于艾宾浩斯遗忘曲线，科学安排 5 轮复习

## 方法论

本助手基于以下已被验证的学习方法：
- 艾宾浩斯遗忘曲线（间隔重复）
- 费曼学习法（以教代学）
- 主动回忆（提取练习）
- 交错练习
- 精细询问
