# 学习方法论参考库

## 1. 艾宾浩斯遗忘曲线（Spaced Repetition）
- 核心：在即将遗忘前复习，记忆效率最高
- 最佳复习间隔：1天→3天→7天→14天→30天
- 实现工具：闪卡 + 复习计划表
- 适用：需要长期记忆的知识（词汇、公式、概念）

## 2. 费曼学习法（Feynman Technique）
- 核心：如果你不能简单地解释一件事，说明你还没真正理解它
- 四步骤：
  1. 选定概念
  2. 用简单语言解释（假装教一个小孩）
  3. 找到解释中的漏洞
  4. 回到原材料补充漏洞
- 适用：理解型知识（原理、机制、框架）

## 3. 主动回忆（Active Recall）
- 核心：主动提取记忆，比被动重读强 50%+
- 方法：先合上材料，尝试默写/口述，再对照检验
- 适用：所有类型的知识

## 4. 交错练习（Interleaving）
- 核心：混合练习不同主题，而非专注单一主题
- 短期感觉更难，长期记忆效果更好
- 适用：有多个子主题的学科（如数学不同题型混合练习）

## 5. 精细询问（Elaborative Interrogation）
- 核心：不断问"为什么是这样？"和"这和我已知的有什么联系？"
- 适用：建立知识网络，防止孤立知识点
