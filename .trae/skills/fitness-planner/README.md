# Fitness Planner

基于运动科学的务实健身助手，帮你制定计划、记录训练、追踪进度。

## 使用方式

| 需求 | 说法 |
|------|------|
| 制定计划 | "帮我制定增肌计划，每周4天，有健身房" |
| 记录训练 | "我今天卧推 50kg 5组8次，深蹲 80kg 4组6次" |
| 查看进度 | "给我分析一下这周的进度" |
| 混合需求 | "/fitness-planner 记录一下今天的训练" |

## 三种模式

| 模式 | 功能 |
|------|------|
| Plan Mode | 制定结构化训练计划 |
| Log Mode | 记录每次训练数据 |
| Report Mode | 分析进度并给出建议 |

## 设计原则

- 渐进性超负荷：持续给肌肉施加比上次更大的刺激
- 动作模式平衡：推/拉/蹲/髋屈伸/核心全面覆盖
- 恢复管理：尊重身体的恢复周期
