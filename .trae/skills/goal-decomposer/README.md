# Goal Decomposer

把你的愿望变成可执行的计划，AI 帮你拆解成一步步的 Milestone。

## 使用方式

说出你的目标即可：

> "我想在 6 个月内学会 Python 数据分析，零基础，每天能学 1 小时"

> "@goal-decomposer 帮我拆解：今年想存 5 万块"

## 支持拆解的目标类型

| 类型 | 示例 |
|------|------|
| 技能类 | 学语言、学编程、学乐器 |
| 物品类 | 买设备、旅行基金 |
| 项目类 | 写书、做 App |
| 健康类 | 减脂、增肌、跑马拉松 |

## 输出内容

每次输出包含：
- 目标卡片（概览）
- Milestone 计划（阶段性拆解）
- 风险提示（帮你避开坑）

## 小贴士

目标越具体，拆解越精准。尽量告诉我截止时间和每天可用时间。
